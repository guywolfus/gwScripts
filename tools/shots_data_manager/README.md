# gwShotsDataManager

## Author:
Created by Guy Wolfus.

## Description:
A handy tool for Maya that allows splitting a scene into several other scenes, mainly envisioned to be used on a layout sequence in order to split it into numbered shots.

## How to Use:
In Maya, use the following Python code to run the tool:
```markdown
from gwScripts.tools import gwShotsDataManager
gwShotsDataManager.run()
```
